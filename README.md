## ⚙️ Getting Started

### Prerequisites

- A web server with PHP support (e.g., XAMPP, WAMP, LAMP, or a live server)
- A modern web browser

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

2. **Move to your web server's root directory:**
   ```bash
   # For XAMPP (Windows)
   mv YOUR_REPO_NAME C:/xampp/htdocs/

   # For LAMP (Linux)
   mv YOUR_REPO_NAME /var/www/html/
   ```

3. **Start your web server** and navigate to:
   ```
   http://localhost/YOUR_REPO_NAME/
   ```

### Static Preview (without PHP)

If you just want to preview the front-end:
- Open `index.html` directly in your browser, or
- Use the VS Code **Live Server** extension.
